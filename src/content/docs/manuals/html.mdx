Good practice to have 
```html
<!DOCTYPE html>
```
at the start of the file (so browsers recognize the file type), in addition to:
```html
<html>

</html>
```
surrounding the full document.

## Title
```html
<title>My Website</title>
```

Sets the title for the page. *Title is shown:*
1. in the browser tab
2. as the title when the page is added to favorites/bookmarks
3. When the page appears in search engines

<Aside type="danger" title="Warning"> The title can *only* be text. </Aside>

## Style Block

```html
<head>
	<style>
		h1 {
			color:blue;
		}
		
		h2{
			color:red;
		}
	</style>
</head>
```

Creates a space to put CSS styling.

## Links

```html
<link rel="stylesheet" href="stylesheet.css">
```

Creates a link between the current page/document and an external source.

*Can link to:*
1.  Stylesheets
2. Favicons (icon shown in browser tab)
3. etc.

See [here](https://www.w3schools.com/tags/tag_link.asp)  for more params.
