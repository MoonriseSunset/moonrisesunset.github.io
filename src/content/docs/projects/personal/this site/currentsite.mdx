This is the 3rd major iteration (or dare I say reincarnation) of my personal site. Lemme break down how this site is set up:

## Site Breakdown

This site along with my main site are hosted within a <a target="_blank" href="https://github.com/MoonriseSunset/moonrisesunset.github.io">single github repository</a> and subsequently served on a single Github Pages Instance, under different directories. Both sites run **Astro** as the framework, with this site specifically utilizing *Starlight* as a further framework.

I used *VSCode* to write the more code heavy sections of this site along with managing source control and npm packages, and *Obsidian* for writing the more text heavy parts.

## History

### 1st Iteration

The first iteration of this site was super simple, just around 5 files, 3 HTML files and 2 CSS files sitting in the root of a github pages repository. I made this iteration of the site in high school while learning how to use an external CSS stylesheet. This old version of the site used a modified form of [Niagara](/projects/personal/niagara), a template I made to just chuck in projects to make them look decent.
